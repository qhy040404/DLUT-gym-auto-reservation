## 特性
- 需要自行部署
- 可通过邮件推送预约结果，以及预约失败原因
- 支持多人（自行添加userdata2）

## 依赖项
- [Python](https://www.python.org/downloads/) 
- requests
  - ```pip install -U requests``` (For Linux)
  - ```py -m pip install -U requests``` (For Windows)

## 使用
- 在```main.py```中修改用户数据
- 运行```py main.py```
